IP Region Flag Privacy Policy

Last updated: 2026-06-07

IP Region Flag is a Chrome extension that shows your current public IP address and approximate IP-based location.

Data used by the extension

The extension requests and displays your public IP address, country, country code, region, city, and last update time.

How the data is used

This data is used only to show your IP address, approximate location, and matching country flag in the extension popup and toolbar icon.

Third-party services

The extension requests IP location details from ip-api.com and loads country flag images from flagcdn.com.

Local storage

The latest IP lookup result is stored locally in Chrome storage so the popup can open quickly and the toolbar icon can show the last detected country.

Data not collected

The extension does not collect browsing history, website content, passwords, authentication data, financial data, health data, personal communications, or files from your device.

Data sharing

The developer does not sell user data and does not use the data for advertising, analytics, profiling, or unrelated purposes.

Contact

For privacy questions, use the publisher contact email listed on the Chrome Web Store listing.
